# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

Please report any security vulnerabilities directly to the project maintainers. Do not open a public issue for security-related concerns.

To report a vulnerability, please email the maintainer or send a private message. You should receive a response within 48 hours. If the issue is confirmed, we will release a patch as soon as possible.
